# Tutorial 1: Prerequisites

Before you start, have these in place:

- **Node.js 20+** (or 22+)  
  - [nodejs.org](https://nodejs.org) or `nvm install 22`

- **Git**  
  - [git-scm.com](https://git-scm.com)

- **Google account**  
  - For Firebase Console and CLI login

- **GitHub account**  
  - For the repo and GitHub Actions

- **A front-end app with a build step**  
  - e.g. React + Vite: `npm run build` outputs to `dist`

Check Node:

```bash
node -v   # should be v20.x or higher
```

Check Git:

```bash
git --version
```

When you’re set, go to [Tutorial 2: Create a Firebase project](02-create-firebase-project.md).
