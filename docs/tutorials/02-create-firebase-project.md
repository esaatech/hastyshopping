# Tutorial 2: Create a Firebase Project (Your “Business”)

Create the Firebase project that will hold your app and Hosting.

## Steps

1. **Open the Firebase Console**  
   [https://console.firebase.google.com](https://console.firebase.google.com)

2. **Create a project**
   - Click **“Add project”** (or **“Create a project”**).
   - Enter a name (e.g. **HastyShopping**).
   - Turn off Google Analytics if you don’t need it (you can enable it later).
   - Click **“Create project”** and wait until it’s ready.

3. **Note the project ID**
   - Open **Project settings** (gear icon).
   - Copy the **Project ID** (e.g. `hastyshopping-2e946`). You’ll use it when linking the CLI and in GitHub.

You now have your “business” in Firebase.
